# TSW Downloader

A fast, modern downloader for **The Secret World**.

## Download

**[Latest Release](https://github.com/EndlessVex/TSW-Modern/releases/latest)** — grab the `.exe` installer and run it.

## What It Does

- Downloads The Secret World at full speed using parallel connections that don't degrade
- Shows honest progress — percentage is accurate, never exceeds 100%
- Resumes where it left off if interrupted
- Launches the original ClientPatcher.exe for game login once the download is complete
- ESC to pause, Space to resume
